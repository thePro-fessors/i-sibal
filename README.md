# i-Sibal: 가볍고 빠른 교실/발표용 AirPlay 수신기

**i-Sibal**은 Raspberry Pi 4B 기반에서 동작하는 독립형 AirPlay 화면 공유 박스입니다. 
복잡한 학교망 연결이나 외부 인터넷 의존 없이, 자체적으로 2.4GHz Wi-Fi 핫스팟을 생성하여 macOS 및 iOS 기기에서 즉시 화면 미러링을 할 수 있도록 설계되었습니다.

특히 발표 상황에서의 "화면 깨짐", "색 번짐", "화면 끊김"을 막기 위해 하드웨어 디코딩과 GStreamer 파이프라인 최적화에 초점을 맞추었습니다.

## 핵심 특징
1. **완벽한 독립형 핫스팟**: 내장 Wi-Fi를 하드웨어 레벨에서 비활성화하고, 안정성이 비교적 높은 단일 채널 2.4GHz USB 동글(RTL8188FTV 등)을 사용하여 자체 망을 구축합니다. `wlan1` 등 무선 인터페이스 이름이 변경되더라도 TUI 환경에서 직관적으로 선택할 수 있습니다.
2. **듀얼 모드 지원 (안정성 vs 저지연)**:
   * **부드러운 재생(안정성) 모드 [기본]**: 발표 중 Wi-Fi 간섭이 발생하더라도 넉넉한 버퍼를 통해 화면 찢어짐이나 끊김을 방지합니다. 프레임 드랍 없는 안정성이 최우선입니다.
   * **저지연(Low-Latency) 모드**: 영상 시청 등 립싱크(A/V Sync)가 중요하거나 실시간 반응성이 필요한 경우, 간섭으로 인한 프레임 스킵을 감수하고 딜레이를 최소화합니다.
3. **색 품질 최적화**: HDMI 픽셀 인코딩 제어 및 BT.709 컬러 리매핑을 통해 "물빠진 색감"과 텍스트 번짐을 해결합니다.
4. **가벼운 운영체제**: 무거운 데스크탑 환경(X11/Wayland) 없이 Raspberry Pi OS Lite USB 부팅 환경에서 KMS(Kernel Mode Setting)를 통해 디스플레이로 직접 화면 출력합니다.
5. **통합 TUI 관리자 지원**: CLI 환경의 불편함을 덜어주기 위해 모든 기능(모드 변경, AP 재설정, 로그 확인)을 통합 제어할 수 있는 그래픽 형태의 터미널 메뉴(`i-sibal.sh`)를 제공합니다.

## 하드웨어 요구사항
* **보드**: Raspberry Pi 4B (4GB) 
* **저장매체**: 고속 USB 플래시 드라이브 또는 USB SSD (OS 64-bit Lite 설치 및 USB 부팅)
* **네트워크**: RTL8188FTV 칩셋 기반 2.4GHz 단일 채널 USB Wi-Fi 동글
* **전원**: 공식 5.1V / 3A 전원 어댑터

> [!WARNING]
> **지원 제한 사항**
> - **HDCP/DRM 미지원**: Netflix, Apple TV+ 등 DRM으로 보호되는 콘텐츠는 화면 미러링으로 재생할 수 없습니다.
> - **초저지연 게이밍 비권장**: 2.4GHz 대역의 물리적 한계로 10ms 이하의 딜레이가 필요한 게임용으로는 부적합합니다.

## 설치 및 시작하기

1. **라즈베리파이 부팅**: Raspberry Pi OS Lite 64-bit를 USB에 구워 부팅합니다.
2. **저장소 복제**:
   ```bash
   git clone https://github.com/your-username/i-Sibal.git
   cd i-Sibal
   ```
3. **설치 스크립트 순차 실행**:
   ```bash
   ./scripts/01_install_deps.sh
   # 2번 스크립트 실행 시 파란색 TUI 화면이 뜨며 SSID와 동글 인터페이스(wlan0, wlan1 등)를 편하게 고를 수 있습니다.
   sudo ./scripts/02_setup_ap.sh
   ./scripts/03_install_uxplay.sh
   ./scripts/04_config_system.sh
   ```
4. **재부팅**: `sudo reboot`
5. **사용**: 설정하신 AP Wi-Fi에 접속한 뒤, 모바일 기기의 AirPlay 미러링 메뉴에서 `i-Sibal`을 선택합니다.

## TUI 통합 관리 메뉴 사용법
설치가 완료된 후에는 개별 스크립트를 찾을 필요 없이 하나의 메인 명령어로 프로젝트 전체를 관리할 수 있습니다.
```bash
sudo ./i-sibal.sh
```
위 명령어를 실행하면 다음과 같은 작업들을 화살표 키로 쉽게 조작할 수 있습니다.
- 안정성 ↔ 저지연 렌더링 모드 즉각 전환
- 핫스팟 SSID 및 비밀번호 간편 재설정
- 현재 USB 동글의 네트워크 인터페이스 맵핑 상태 확인
- AirPlay 렌더링 실시간 오류 로그 모니터링
